# API Reference

Base URL (local dev): `http://localhost:8000`

Interactive Swagger docs are also auto-generated by FastAPI at `/docs` and
ReDoc at `/redoc` whenever the backend is running.

## Health

| Method | Path          | Description          |
|--------|---------------|-----------------------|
| GET    | `/`           | Service info/health   |
| GET    | `/api/health` | Liveness check         |

## Stadium Copilot

### `POST /api/chat`

```json
// Request
{
  "message": "Where is Gate B?",
  "language": "en",
  "session_id": "optional-client-generated-id"
}
```

```json
// Response
{
  "reply": "Gate B is on the East side of the stadium...",
  "sources": [
    { "title": "Gate B Location", "content": "...", "score": 0.42 }
  ],
  "used_knowledge_base": true,
  "language": "en"
}
```

`language` must be one of: `en`, `hi`, `es`, `fr`, `ar`, `pt`.

## Crowd Intelligence

| Method | Path                        | Description                                  |
|--------|------------------------------|-----------------------------------------------|
| GET    | `/api/crowd/snapshot`        | Simulated occupancy across gates/washrooms/food/parking |
| GET    | `/api/crowd/recommendations` | AI-generated operational recommendations      |

## Emergency Assistant

### `POST /api/emergency/report`

```json
// Request
{
  "incident_type": "medical",
  "location": "Section 14, near Gate C",
  "description": "A fan has collapsed and needs medical attention."
}
```

```json
// Response
{
  "incident_id": "INC-4F2A9B7C",
  "summary": "...",
  "immediate_actions": ["...", "..."],
  "priority": "high",
  "suggested_evacuation_route": "...",
  "volunteers_notified": ["Volunteer #12 (Gate A)", "..."]
}
```

`incident_type` must be one of: `medical`, `lost_child`, `fire`,
`suspicious_object`, `other`.

## Accessibility

### `POST /api/accessibility/route`

```json
// Request
{ "start": "Gate A", "destination": "Seat B24" }
```

```json
// Response
{
  "route_steps": ["...", "..."],
  "estimated_minutes": 8,
  "accessible_features": ["Ramp access", "Elevator access (no stairs)", "..."]
}
```

## Error format

All errors return:

```json
{ "detail": "Human-readable message" }
```

with an appropriate HTTP status code (`422` for validation errors, `500` for
unexpected server errors).
